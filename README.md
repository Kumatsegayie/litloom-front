# Litloom Frontend

Frontend documentation has been consolidated into the root project documentation:

- `../PROJECT_DOCUMENTATION.md`

Quick start:

```bash
npm install
npm start
```

Production build:

```bash
npm run build
```

Environment reference:
- Use `frontend/.env.example` for production.
- In local development, `frontend/.env.development` is configured for proxy mode (`REACT_APP_STRAPI_URL=`).
